## Color Scheme

- **Primary (Orange)**: `#F97316` - Buttons, selected items, prices
- **Gray**: Background for info boxes
- **White**: Main background, cards
- **Green**: Checkmarks, success messages
- **Red**: Delete buttons, error messages

## Responsive Design

The layout is fully responsive:
- **Desktop**: Side-by-side layout (image left, details right)
- **Tablet**: Stacked layout, full width components
- **Mobile**: Single column, touch-friendly buttons

## Interactive Elements

1. **Checkboxes**: Click anywhere on patch card
2. **Dropdown**: Click to open, smooth animation
3. **Price**: Updates immediately on selection
4. **Hover**: Border color changes on patch cards
5. **Add to Cart**: Disabled until size selected
